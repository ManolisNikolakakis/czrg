# Cazarog (Working Title)

A top-down action roguelike with a cozy aesthetic, currently in early prototype stage. Built in Python and Pygame.

## About

Cazarog is a top-down action roguelike featuring procedurally generated dungeons, real-time combat, and a warm pixel art visual style. The goal is a game that feels mechanically deep and replayable — think Hades' satisfying combat loop wrapped in the cozy aesthetic of something like Garden Story.

Current prototype features:
- Top-down 8-directional movement and combat
- Procedurally generated dungeon floor
- Multiple enemy types with distinct behaviours
- Health items and powerups
- Simple progression loop with permadeath

## Status

🚧 **Early prototype** — core systems in place, actively iterating.

Planned release on [itch.io](https://itch.io) when the game reaches a playable, polished state.

## Human-Made Art & Music

All pixel art and music in this game are original, human-made works by the developer. No AI-generated art or music is used at any point in this project.

## Development Approach

This project is an experiment in **agentic AI-assisted development**. The code is written with the help of [Claude Code](https://claude.ai), Anthropic's agentic coding tool, as a way to explore and learn modern AI-assisted software development workflows.

Game design, creative direction, all artwork, and all music are entirely human-made. Claude Code serves as a coding assistant — the creative vision and decisions are the developer's own.

## Tech Stack

- **Language:** Python 3
- **Framework:** Pygame
- **Version Control:** Git / GitHub

## Running the Game

Make sure you have Python 3 and Pygame installed:

```bash
pip install pygame
```

Then run:

```bash
python cozy_roguelike.py
```
