# Cazarog

A top-down action roguelike currently in early prototype stage. Built in Python and Pygame.

## About

Cazarog is a top-down action roguelike featuring procedurally generated dungeons, real-time combat, and a pixel art visual style. The goal is a game that feels mechanically deep and replayable. The game is inspired by Wreck-It Ralph and Hades.

Current prototype features:
- Top-down 8-directional movement and combat
- 9 rooms to go through in a dungeon-crawler
- A hub to hang out with the characters from the game
- 3 unique fighters to choose from
- 5 pre-boss minigames
- 3 unique bosses
- Health items and powerups

## Status

🚧 **Early prototype** — core systems in place, actively iterating.

Planned release on [itch.io](https://itch.io) when the game reaches a playable, polished state.

## All creative aspects of the game are human made

All dialogue, writing, game design pixel art and music in this game are original, human-made works by the developer. No AI-generated art or music is used at any point in this project.

## Development Approach

This project is an experiment in **agentic AI-assisted development**. The code is written with [Claude Code](https://claude.ai) to explore and learn modern AI-assisted software development workflows.
Claude Code serves as a coding assistant — the creative vision and decisions are the developer's own.

## Tech Stack

- **Language:** Python 3
- **Framework:** Pygame

## Running the Game

Make sure you have Python 3 and Pygame installed:

```bash
pip install pygame
```

Then run:

```bash
python cozy_roguelike.py
```
